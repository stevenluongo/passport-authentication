## Local Installation

1. Clone the repo
   ```sh
   git clone https://github.com/binolt/next.js-passport-auth.git
   ```
2. Install dependencies
   ```sh
   npm install
   ```
3. Start the live server
   ```sh
   npm run dev
   ```
